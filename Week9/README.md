# What's New This Week

# Book Library Management System

A complete library management application with full CRUD operations and persistent storage.

**Features:**
- Add books with title, author, genre, and page count
- Search library by title or author
- Delete books from collection
- View entire library catalog
- Duplicate prevention (same title + author check)
- Data persistence using pickle

# Marks_to_grade V9.0
Enhanced version with:
- Student management (add/delete/update/search)
- Automatic grade calculation
- Statistics: highest/lowest marks, average, pass/fail rates
- Grade distribution analysis
- Input validation and duplicate prevention

# Data Persistence

All applications now use **pickle** for permanent data storage:
- `library.pck` - Book collection
- `students.pck` - Student database
- `shopping_list_file.pck` - Shopping items
- `cars.pck` - Car rental fleet
- `holidays.pck` - Travel plans
- `recipes.pck` - Recipe collection
